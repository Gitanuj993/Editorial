<h3> Headings in Markdowns </h2>
```Text
Heading like H1, H2 ... H6 in markdown by using '#' then a sapce

# heading 1
## heading 2
### heading 3
#### heading 4
##### heading 5
###### Heading 6
```

### List in Markdown 

```Text

List followed by numbers and -   , + or any symbol

1. list1
2. list2
3. list3
	- sublist1
	- sublist2
	- sublist3
		- sub-sub-list1
		- sub-sub-list2


We can also use <ol> <li> List1 </li> </ol> or <ul> <li> list1 </li> </ul>
```
